# UI 第二轮优化说明

本轮主要针对用户反馈继续优化了答题体验与弹窗交互，未改动核心游戏逻辑。

## 本轮新增 / 调整

1. **替考答题界面增强**
   - 答对显示 `😊`，答错显示 `😢`。
   - 提交后即时反馈颜色更明显。
   - 选项提交后会锁定，避免误操作。

2. **替考结束反馈弹窗**
   - 替考完成后新增结果弹窗。
   - 展示答对题数、正确率、成绩提升、金币消耗。

3. **更雅观的措辞**
   - “转学变现”按钮改为：**协助转学发展**。
   - 相关确认文案、日志文案、提示文案同步优化。
   - 已转学学生状态文案调整为：**已转学**。

4. **推进阶段弹窗美化**
   - 生活费、考试结果、随机事件、错误提示、结局提示改为更丰富的卡片式信息弹窗。
   - 加入 emoji 和层次化文案，减少单调感。

5. **点击学生后的反馈更明显**
   - 学生表下方新增“即时反馈卡”。
   - 选中学生后会高亮显示：当前状态、三维属性、参考价、培养建议。
   - 表格选中态高亮也进一步增强。
